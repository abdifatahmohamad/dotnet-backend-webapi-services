When installing EF and versioning issues, do the following:

Option A (recommended): Use a local tool manifest (best for projects)

This avoids global-tool corruption and keeps version pinned per repo.

From the folder that contains your .csproj (HospitalSystem.Api):

dotnet new tool-manifest

dotnet tool install dotnet-ef --version 8.0.11


Verify:

dotnet ef --version
